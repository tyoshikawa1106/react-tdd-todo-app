React × TDD × Todo Apps

## Table of Contents
- [Requirements](#requirements)
- [Usage](#usage)

## Requirements
- Node v6.0.0 ~
- nodemon v1.9.2
- mocha v2.5.3

## Usage
Run Apps
```
$ npm install
$ npm webpack --watch
$ nodemon
```

Run Test
```
$ cd react
$ cd modules
$ cd spec
$ mocha --require setup.js *.spec.js
```
